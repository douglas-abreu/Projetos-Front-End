Básicos de HTML

Olá pessoal! Como um método de aprendizado e pra ajudar aqueles que possuem interesse em aprender HTML, estarei criando este manual/documentação para me auxiliar em meus estudos de fixação e quem sabe, também poder te ajudar aprender os básicos dessa famosa linguagem de marcação.

Esqueleto de uma página HTML

O esqueleto de uma página HTML é constituído pelas tags:

<html>

<head>

</head>

<body>

</body>

</html>


As tags "<tag>" indica o início da tag, e "</tag>" juntamente com uma barra "/" estaremos indicando o fim da tag.

Inserindo o título da página

Sabe aquele nome que fica lá no topo das páginas em seus navegadores informando o nome do website? Também é conhecido por "título", e para que você atribua um título para a sua página, com a tag title no head da sua página aplique a seguinte informação:

<title>nome do título</title>

No meu caso, atribuí o nome de "Teste HTML".

Criando um Header

Para estarmos inserindo um Heading à página, com a tag "h1" isso será possível, o numeral ao lado do "h" informa o nível de importância do Header, quanto maior o número menor será a importância, o título para essa parte foi implementado com o a importância de nível 3. Veja um exemplo:

<h3>texto</h3>

Implementando um link

Provavelmente, durante a sua criação terá interesse de estar aplicando um hypertext em uma palavra ou frase como este link para o youtube. Com a tag "a" e atribuindo à tag href. Veja o exemplo:

<a href= "url">palavra ou frase que receberá esse hyperlink</a>

Parágrafos

Com a tag "p" torna-se possível a criação de um parágrafo. "Mas pra que parágrafo se posso estar fazendo um texto direto no body? Está relacionada ao fato de que ele possibilita a definição dos atributos id e class para cada parágrafo e assim é possível formatá-los uniformemente com CSS. Exemplo:

<p>texto</p>

Formatações simples sem CSS

Para formatações simples sem a necessidade de CSS, é uma pequena lista de comandos que podem sem realizados para tal ação.

<b>: Define texto em negrito;
Ex: <b>texto</b>

<u>: Define texto sublinhado;
Ex: <u>texto</u>

<s> Define texto riscado;
Ex: <s>texto</s>

<i> Define texto em itálico;
Ex: <i>texto</i>

<small> Define texto em uma fonte de tamanho menor;
Ex: <small>texto</small>

<sub> Define texto subescrito;
Ex: <sub>texto</sub>

<sup> Define texto superescrito;
Ex: <sup>texto</sup>

Quebra de linha

Quando você estiver realizando a implementação de um texto em uma tag de parágrafo, ele ignora as quebras de linhas que você realiza no código, para que o HTML entenda que você quer uma quebra de linha é necessário estar implementando a tag "br". Veja o exemplo:

Sem quebra de linha

Com
quebra
de
linha

Estrutura do exemplo a cima: <p>Com<br>quebra<br>de<br>linha</p>

Listas

Existem três tipos de listas: ordenadas, não ordenadas e de definição. Cada tipo de lista é delimitada por uma tag diferente:

  • <ol>...</ol>: para listas ordenadas (ordered list). Essas listas são utilizadas quando se quer numerar algo;

  • <ul>...</ul>: para listas não ordenadas (unordered list). Essas listas são utilizadas quando se quer apenas colocar um marcador em seus itens, mas sem numeração;

  • <dl>...</dl>: para listas de definição (definition list). Essas listas são usadas quando, além de topificar, ainda se deseja especificar uma definição para cada tópico.

Tabelas

Tabelas são definidas com a tag <table>…</table>. Elas são divididas em linhas, delimitadas pela tag <tr>...</tr>, e colunas, delimitadas pela tag <td>...</td>, nas quais se determina o conteúdo das células. Esse conteúdo pode ser texto, imagens, listas, formulários ou outros elementos.

O elemento table pode conter os atributos:

  • Border: indica qual o tipo de borda da tabela (se não for definido, nenhuma borda aparecerá);
  • Width: indica a largura da tabela (ele pode ser dado em percentual);

Exemplo:

Título 1 Título 2
Item linha 1 e coluna 1 Item linha 1 e coluna 2
Item linha 2 e coluna 1 Item linha 2 e coluna 2

Estrutura do exemplo a cima:

<table border="1px" width="40%">
<tr>
<th>Título 1</th>
<th>Título 2</th>
</tr>
<tr>
<th>Item linha 1 e coluna 1</th>
<th>Item linha 1 e coluna 2</th>
</tr>
<tr>
<th>Item linha 2 e coluna 1</th>
<th>Item linha 2 e coluna 2</th>
</tr>
